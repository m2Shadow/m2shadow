mac mysql修改密码以及开通访问权限

修改密码

关闭数据库

mysql.server stop

进入到 mysql 所在的 bin 目录

cd /usr/local/Cellar/mysql/5.7.17/bin

具体还是要看自己电脑里的路径

开通管理员权限

sudo su

禁止 mysql 验证功能

./mysqld_safe --skip-grant-tables &

输入命令

./mysql

输入命令

FLUSH PRIVILEGES;

注意：最后的 ; 不能少，不然会报语法错误。

修改密码

SET PASSWORD FOR 'root'@'localhost' = passwd('你的新密码');

退出

\q

启动 mysql(可忽略)

mysql.server start

输入以下命令

/usr/local/Cellar/mysql/5.7.17/bin/mysql -u root -p

接着输入你刚刚设置的密码就可以启动 mysql 了。

访问数据库受限

问题描述：别人无法访问到你的数据库。

设置对外开放

  1. 首先点击 select database ，选择 mysql。
  2. 找到 user 表
  3. 点击 content
  4. 把 host 下的 local 改成 %
  5. 终端下输入 /usr/local/Cellar/mysql/5.7.17/bin/mysql -u root -p(根据自己的路径)
  6. 输入 FLUSH PRIVILEGES;
  7. 输入 \q

如果这个时候依旧发现无法访问，那么就检查下本机的防火墙有没有关闭了。
