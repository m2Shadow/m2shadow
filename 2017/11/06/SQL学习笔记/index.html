SQL学习笔记

  • SELECT * FROM Websites; 读取数据表的信息。
  • SELECT name,country FROM Websites;
  • SELECT DISTINCT country FROM Websites;去掉 “country” 列重复值
  • SELECT * FROM Websites WHERE country=’CN’;取国家为 “CN” 的所有网站
  • SELECT * FROM Websites WHERE country=’CN’ AND alexa > 50;选取国家为 “CN” 且alexa排名大于 “50” 的所有网站
  • SELECT * FROM Websites WHERE country=’USA’ OR country=’CN’;选取国家为 “USA” 或者 “CN” 的所有客户
  • SELECT * FROM Websites WHERE alexa > 15 AND (country=’CN’ OR country=’USA’);
  • SELECT * FROM Websites ORDER BY alexa;按照 “alexa” 列排序
  • SELECT * FROM Websites ORDER BY alexa DESC;按照 “alexa” 列降序排序
  • SELECT * FROM Websites ORDER BY country,alexa;按照 “country” 和 “alexa” 列排序
  • INSERT INTO Websites (name, url, alexa, country) VALUES (‘百度’,’https://www.baidu.com/','4','CN‘); 向 “Websites” 表中插入一个新行
  • UPDATE Websites SET alexa=’5000’, country=’USA’ WHERE name=’菜鸟教程’;
  • DELETE FROM Websites WHERE name=’百度’ AND country=’CN’;
  • SELECT * FROM Websites LIMIT 2; 选取头两条记录
  • SELECT TOP 50 PERCENT * FROM Websites; 选取前面 50% 的记录
  • SELECT * FROM Websites WHERE name LIKE ‘G%’; 以字母 “G” 开始的所有客户
  • SELECT * FROM Websites WHERE name LIKE ‘%k’; 以字母 “k” 结尾的所有客户
  • SELECT * FROM Websites WHERE name LIKE ‘%oo%’; name 包含模式 “oo” 的所有客户
  • SELECT * FROM Websites WHERE name NOT LIKE ‘%oo%’;
  • SELECT * FROM Websites WHERE name LIKE ‘_oogle’;一个任意字符开始
  • SELECT * FROM Websites WHERE name REGEXP ‘^[GFs]’;以 “G”、”F” 或 “s” 开始的所有网站
  • SELECT * FROM Websites WHERE name IN (‘Google’,’菜鸟教程’);选取 name 为 “Google” 或 “菜鸟教程” 的所有网站
  • SELECT * FROM Websites WHERE alexa BETWEEN 1 AND 20;alexa 介于 1 和 20 之间的所有网站
  • SELECT * FROM Websites WHERE alexa NOT BETWEEN 1 AND 20;
  • SELECT * FROM Websites WHERE (alexa BETWEEN 1 AND 20) AND NOT country IN (‘USA’, ‘IND’);
  • SELECT * FROM Websites WHERE name BETWEEN ‘A’ AND ‘H’;’A’ 和 ‘H’ 之间字母开始的所有网站
  • SELECT * FROM access_log WHERE date BETWEEN ‘2016-05-10’ AND ‘2016-05-14’;
  • SELECT name AS n, country AS c FROM Websites; 列的别名实例
  • SELECT name, CONCAT(url, ‘, ‘, alexa, ‘, ‘, country) AS site_info FROM Websites;我们把三个列（url、alexa 和 country）结合在一起，并创建一个名为 “site_info” 的别名
  • SELECT w.name, w.url, a.count, a.date FROM Websites AS w, access_log AS a WHERE
    a.site_id=w.id and w.name=”菜鸟教程”;
  • SELECT Websites.id, Websites.name, access_log.count, access_log.date FROM Websites INNER JOIN access_log ON Websites.id=access_log.site_id;
  • SELECT country FROM Websites UNION SELECT country FROM apps ORDER BY country;从 “Websites” 和 “apps” 表中选取所有不同的country（只有不同的值）
  • SELECT country FROM Websites UNION ALL SELECT country FROM apps ORDER BY country;有重复的值
  • INSERT INTO Websites (name, country) SELECT app_name, country FROM apps;复制 “apps” 中的数据插入到 “Websites” 中
  • CREATE DATABASE my_db; 创建一个名为 “my_db” 的数据库

在 SQL 中，我们有如下约束:

  • NOT NULL - 指示某列不能存储 NULL 值。
  • UNIQUE - 保证某列的每行必须有唯一的值。
  • PRIMARY KEY - NOT NULL 和 UNIQUE 的结合。确保某列（或两个列多个列的结合）有唯一标识，有助于更容易更快速地找到表中的一个特定的记录。
  • FOREIGN KEY - 保证一个表中的数据匹配另一个表中的值的参照完整性。
  • CHECK - 保证列中的值符合指定的条件。
  • DEFAULT - 规定没有给列赋值时的默认值。

CREATE INDEX 语句用于在表中创建索引。

AUTO_INCREMENT 的开始值是 1，每条新记录递增 1

  • DROP TABLE table_name; 用于删除表
  • DROP DATABASE database_name 删除数据库
  • ALTER TABLE table_name DROP INDEX index_name
  • ALTER TABLE table_name ADD column_name datatype 在表中添加列
  • ALTER TABLE table_name DROP COLUMN column_name 删除
  • ALTER TABLE Persons ADD DateOfBirth date; 添加一个名为 “DateOfBirth” 的列
  • ALTER TABLE Persons ALTER COLUMN DateOfBirth year 改变 “Persons” 表中 “DateOfBirth” 列的数据类型
  • ALTER TABLE Persons DROP COLUMN DateOfBirth; 删除
